# 🚀 KL Attendance Calculator

A modern and aesthetic attendance analysis web app inspired by KL University attendance systems.
Built using **HTML, CSS, and JavaScript** with premium UI, smooth animations, LTPS weighted calculations, and dual themes.

---

# ✨ Features

- 📊 Simple Attendance Calculator
- 🔬 LTPS Weighted Attendance Calculator
- 🎯 Attendance Eligibility Prediction
- ⚠️ Condonation & Not Eligible Detection
- 🌌 Dual Themes
  - 😎 Boys Cyberpunk Theme
  - ✨ Girls Dreamy Aesthetic Theme
- 💫 Animated Background Effects
- 🔥 Glassmorphism UI
- 🎡 Animated Progress Ring
- 📱 Fully Responsive Design
- 🚀 Smooth Transitions & Interactive Effects

---

# 🧠 LTPS Logic

The LTPS calculator follows weighted calculations:

| Component | Weight |
|-----------|--------|
| Lecture   | 100 |
| Tutorial  | 25 |
| Practical | 50 |
| Skilling  | 25 |

### Important Logic
- Empty fields are automatically ignored.
- Only entered fields are considered in calculations.
- Dynamic weighted average calculation.

# 🎯 Attendance Status

| Attendance | Status |
|------------|--------|
| ≥ 85% | ✅ SAFE |
| 75% – 84.99% | ⚠️ CONDONATION |
| < 75% | ❌ NOT ELIGIBLE |
